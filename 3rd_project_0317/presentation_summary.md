# 발표용 요약본

## 1. 주제

`clean_dataset.py`를 이용한 제조 데이터 품질 점검 및 전처리

## 2. 발표 핵심 내용

- 이 코드는 원본 CSV 데이터를 불러와 정제된 새 CSV 파일로 저장하는 전처리 코드이다.
- 목적은 분석이나 머신러닝 전에 데이터 품질을 먼저 확보하는 것이다.
- 코드에서는 유일성, 완전성, 유효성, 일관성, 정확성의 5가지 기준으로 데이터를 검사한다.
- 중복 데이터는 제거하고, 핵심 센서값에 결측치가 있는 행은 제거한다.
- 물리적으로 불가능한 시간 데이터와 논리적으로 모순된 데이터도 제거한다.
- 마지막으로 비정상적인 배럴 온도 데이터를 제거하여 더 신뢰할 수 있는 데이터셋을 만든다.

## 3. 1분 발표 예시

안녕하세요.  
이번 실습에서는 `clean_dataset.py` 코드를 통해 제조 공정 데이터를 정제하는 과정을 확인했습니다.  
이 코드는 원본 CSV 데이터를 불러온 뒤, 중복 데이터 제거, 결측치 제거, 물리적으로 불가능한 값 제거, 논리적 모순 제거, 비정상 온도 제거를 수행합니다.  
특히 데이터 품질을 유일성, 완전성, 유효성, 일관성, 정확성의 다섯 가지 기준으로 나누어 확인했다는 점이 중요합니다.  
이를 통해 데이터는 단순히 많이 모으는 것보다, 분석 전에 신뢰할 수 있는 상태로 정제하는 과정이 반드시 필요하다는 점을 배울 수 있었습니다.  
결과적으로 이 코드는 머신러닝이나 통계 분석에 앞서 수행해야 하는 데이터 전처리의 기본 원리를 잘 보여주는 예제라고 생각합니다.

## 4. 짧은 발표용 키워드

- 데이터 전처리
- 데이터 품질 관리
- 중복 제거
- 결측치 제거
- 이상치 제거
- 논리적 모순 제거
- 분석 신뢰도 향상
