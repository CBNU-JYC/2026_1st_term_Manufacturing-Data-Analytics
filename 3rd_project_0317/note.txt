제공해주신 코드는 사출성형 공정 데이터를 정제하는 아주 훌륭한 로직을 담고 있네요! **Anaconda(아나콘다)**를 활용하여 이 코드를 실행하기 위한 환경 구축부터 실행까지의 과정을 단계별로 상세히 안내해 드릴게요.

---

## 1단계: Anaconda 설치하기
이미 설치되어 있다면 이 단계는 건너뛰셔도 됩니다.

1.  [Anaconda 공식 홈페이지](https://www.anaconda.com/download)에 접속하여 OS(Windows/Mac/Linux)에 맞는 설치 파일을 다운로드합니다.
2.  설치 프로그램을 실행하고, 기본 설정을 유지하며 설치를 진행합니다.
    * **주의:** 설치 중 'Add Anaconda to my PATH' 옵션은 체크하지 않는 것이 권장됩니다 (대신 'Anaconda Prompt'를 사용하면 됩니다).

---

## 2단계: 프로젝트 폴더 및 데이터 준비
코드가 정상적으로 동작하려면 데이터 파일이 올바른 위치에 있어야 합니다.

1.  컴퓨터의 원하는 위치에 새 폴더를 만듭니다 (예: `C:\my_project`).
2.  해당 폴더 안에 다음 두 파일을 넣습니다.
    * **`labeled_data.csv`**: 분석할 원본 데이터 파일
    * **`clean_data.py`**: 위에서 작성하신 파이썬 코드를 메모장이나 텍스트 에디터에 복사하여 이 이름으로 저장하세요.



---

## 3단계: 가상환경 구축 및 라이브러리 설치
Anaconda는 프로젝트별로 독립된 환경(가상환경)을 만드는 기능을 제공합니다. 이를 통해 라이브러리 충돌을 방지할 수 있습니다.

1.  Windows 시작 메뉴에서 **Anaconda Prompt (anaconda3)**를 찾아 실행합니다.
2.  아래 명령어를 한 줄씩 입력하여 전용 가상환경을 만들고 활성화합니다.
    ```bash
    # 1. 'data_env'라는 이름의 가상환경 생성 (파이썬 3.10 버전)
    conda create -n data_env python=3.10 -y

    # 2. 가상환경 활성화
    conda activate data_env
    ```
3.  코드 실행에 꼭 필요한 **pandas** 라이브러리를 설치합니다.
    ```bash
    pip install pandas
    ```

---

## 4단계: 코드 실행하기
이제 준비가 끝났습니다! 저장한 코드를 실행해 봅시다.

1.  Anaconda Prompt에서 코드가 저장된 폴더로 이동합니다. (경로는 본인의 폴더 위치로 수정하세요)
    ```bash
    cd C:\my_project
    ```
2.  파이썬 파일을 실행합니다.
    ```bash
    python clean_data.py
    ```

---

## 5단계: 결과 확인
코드가 성공적으로 실행되면 터미널 창에 각 단계별(유일성, 완전성, 유효성 등) 검증 결과가 출력됩니다.

* 최종적으로 폴더 안에 **`labeled_data_clean.csv`** 파일이 새로 생성되었는지 확인하세요.
* 이 파일이 바로 모든 결측치와 논리적 오류가 제거된 **'골든 데이터셋(Golden Dataset)'**입니다.

---

### 💡 팁: 코드를 수정하거나 데이터를 시각적으로 보고 싶다면?
명령 프롬프트가 익숙하지 않다면, Anaconda 설치 시 함께 제공되는 **Jupyter Notebook**을 사용해 보세요.

1.  `conda activate data_env` 상태에서 `pip install jupyter`를 입력해 설치합니다.
2.  `jupyter notebook`을 입력하면 웹 브라우저가 열리며, 여기서 코드를 한 줄씩 실행하고 데이터프레임을 표 형태로 바로 확인할 수 있어 매우 편리합니다.

**혹시 실행 중에 `FileNotFoundError` 같은 오류가 발생하거나, 특정 라이브러리가 더 필요하신가요? 말씀해 주시면 바로 해결해 드릴게요!**
